# Projecte-Django
Projecte de Django

Aquest es un projecte per fer una aplicacio web utilitzant el framework Django, on el principal objectiu de la web es una xarxa de blogs de programacio, on pots veure diferents noticies i tutorials.

Instal·lació ràpida:

1. Primer ens hem de descarregar els arxius en .zip
2. Hem de preparar un entorn virtual en una carpeta amb la comanda 'python -m venv env_site'
3. Ara activem l'entor virtual amb la comanda '.\env_site\Scripts\activate.ps1'
4. Despres instal·lem les depencencies 'pip install django'
5. Per ultim descomprimim el zip en la mateixa carpeta
